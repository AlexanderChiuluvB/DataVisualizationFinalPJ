# DataVisualizationFinalPJ
大数据学院数据可视化期末PJ

我们主要是开发了一个GUI，借助Face++的人脸特征点识别API
实现人脸融合，换脸以及特征脸功能。
打包后有个小bug，就是图片素材一定要放在与py文件相同的文件夹才能够使用

具体原理参照ppt与报告
